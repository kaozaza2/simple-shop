# Simple-Shop

Simple-Shop พัฒนาโดยใช้ Lumen framework.

# วิธีติดตั้งใช้งาน

ความต้องการระบบ
* PHP >= 7.3
* Composer
    * วิธีติดตั้ง composer ได้ที่นี่ [https://getcomposer.org](https://getcomposer.org)

วิธีใช้งาน
```
git clone https://github.com/kaozaza2/simple-shop.git
cd simple-shop
composer install --no-dev
```

Local serve
```
php -S localhost:8000 -t public
```
